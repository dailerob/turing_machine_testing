# turing_machine_testing
Repository to generate and simulate Turing machines.

## Running scripts

Use Anaconda Python, or the virtual environment in the sibling directory `automata_distance`:

- **Anaconda** (if on PATH): `python generative_dense_chain_timeseries.py`
- **Anaconda by path** (Windows): `& "$env:USERPROFILE\anaconda3\python.exe" generative_dense_chain_timeseries.py`
- **automata_distance venv** (if present): `..\automata_distance\.venv\Scripts\python.exe generative_dense_chain_timeseries.py`
